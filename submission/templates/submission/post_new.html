Descrição

Resolva o problema de Caxeiro Viajante com um algoritmo aproximado. Este algoritmo deve ser uma 2-aproximação para o problema. Ou seja, o caminho encontrado pode ser até duas vezes maior que a melhor resposta.

Seu algoritmo deve resolver o problema em tempo polinomial e usando a linguagem de programação Python 3.

Divirta-se.

Entrada

A entrada consiste primeiramente de uma linha com um inteiro V, o qual indica o número de vértices no grafo. Nas V linhas seguintes, é dada uma matriz V x V que representa o grafo. Por exemplo: 

4
0 1 10 4
1 0 2 20
10 2 0 3
4 20 3 0

Saída

Consiste em V + 1 inteiros separados por espaços, representando um caminho que respeite a 2-aproximação em uma única linha. Por exemplo:

0 1 2 3 0

Que é um dos caminhos para o grafo acima que respeita a 2-aproximação.
